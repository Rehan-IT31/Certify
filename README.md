CertiFy – Blockchain Certificate Verification System

CertiFy is a full-stack web application that enables secure issuance and verification of digital certificates using blockchain concepts. The system ensures that certificates are tamper-proof, verifiable, and permanently recorded.

🚀 Features

✔ Issue Certificates (PDF upload)
✔ SHA-256 Hash Generation for uniqueness
✔ Blockchain Integration using Blockfrost API
✔ Instant Certificate Verification
✔ Tamper Detection (hash mismatch detection)
✔ Dashboard to view issued certificates
✔ File validation (PDF only, max 1MB)
✔ Clean and professional
